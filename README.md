##tal_kafka

A wrapper cookbook to install nginx using the community cookbooks.

###Platforms

Debian, Ubuntu
Red Hat/CentOS/Scientific (6.0+ required) - "EL6-family"

###Tested on:

Ubuntu 14.04
Centos 7.1

###Usage

On systems that doesnt have an installed nginx, use recipe[tal_nginx::default]
